# KICKR Gears

A lightweight macOS app that displays your WAHOO KICKR bike's gear selection as a floating HUD overlay.

![KICKR Gears Screenshot](https://github.com/booxdk/KickrGears/blob/main/screenshots/screenshot1.jpeg)

## Features

- 🚴 Real-time gear display (front 1-3, rear 1-13)
- 🎯 Always-on-top floating window
- 🎨 Customizable scale and transparency
- 🔗 Automatic Bluetooth connection
- 💻 Native macOS app

## Download

**[Get the latest version](https://github.com/booxdk/KickrGears/tree/main/download)**

**Requirements:** macOS 13.0 (Ventura) or later

## Quick Start

1. Power on your KICKR bike
2. Launch KICKR Gears
3. Position the window over Zwift (use windowed mode)
4. Right-click → "Lock Position" when happy with placement
5. Ride!
